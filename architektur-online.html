🚀 Umsetzung in 7 Schritten
1
GitHub Repo anlegen
Neues privates Repo wochenplaner im bestehenden GitHub-Account (hjhan). index.html hochladen.
2
.bat Skript kopieren
Deploy-Skript vom Panzerspiel kopieren, Pfad anpassen. Doppelklick = deployed.
3
Netlify verbinden
Im bestehenden Netlify-Account neues Projekt, GitHub-Repo wählen → sofort live.
4
Supabase Projekt
Im bestehenden Account neues Projekt wochenplaner (EU Frankfurt) + 5 Tabellen anlegen.
5
SDK einbinden
Supabase JS-SDK per CDN in index.html + URL und Anon-Key aus dem Supabase-Dashboard.
6
localStorage ersetzen
~15 Stellen im Code von localStorage auf Supabase umstellen. Echtzeit-Subscriptions aktivieren.
7
Als App installieren
PWA-Manifest → auf Handys als App speicherbar. Icon + Offline-Modus inklusive.
